FSM with lifecycle hooks

fsm Pipeline with Integer
  Idle    --start-->   Setup
  Setup   --init!-->   Ready
  Ready   --poll?-->   Ready
  Ready   --process--> Done
  *       --abort-->   Failed
  @timer 5000
  on_transition
    (:idle,  :start,   _payload, data) -> %[:ok, :setup,    data]
    (:ready, :process, _payload, data) -> %[:ok, :done,     data]
  on_enter
    (:ready, _state) -> :ok
    (_,      _state) -> :ok

Dependent types and typed holes

mod HolesDemo
  ### Hole-driven development.
  Holes surface in type errors so you can fill them in incrementally:
    ?name -- named, carries a goal type
    ??    -- anonymous, numbered ?_1, ?_2, ...
  ###

  fn double(n: Int) -> Int = n * 2

  fn square_sketch(n: Int) -> Int = ?body

  fn main() -> Int = double(square_sketch(5))
